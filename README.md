# leperssay
leperssay is a configurable talking Julien Lepers.

## Requirements

### Ubuntu/Debian

```bash
sudo apt update && sudo apt -y install gcc make libncurses5-dev
```

### Fedora/Red Hat

```bash
sudo yum update && sudo yum -y install gcc make ncurses-devel
```

## Installation

```bash
make
sudo make install
```

## Usage

```bash
leperssay "C'EST NON"
```
